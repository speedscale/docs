# http\_queryparam

### Purpose

http\_queryparam extracts a query parameter by name.

### Usage

```
"type": "http_queryparam",
"config": {
    "name": "<parameter>"
}
```

**name** - the name of the query parameter to extract

### Example

```
"type": "http_queryparam",
"config": {
    "name": "filter"
}
```
